Resumen

  • Un identificador es el nombre que tiene un elemento de un programa y que no se puede repetir en ciertas partes del código.

    Un identificador debe cumplir con las siguientes reglas:

    • Empezar con una letra (a – z o A – Z), guion bajo (_) o $.

    • Los caracteres siguientes pueden ser cualquiera de los anteriores o dígitos (0 – 9).

  • Una constante es similar a una variable, pero su valor no puede cambiar una vez asignado.

  • Las variables o constantes declaradas en el mismo espacio no pueden tener el mismo nombre.

  • Cuando tratas de usar una variable o una constante no definida, obtienes un mensaje de error y se aborta la ejecución del programa.

  • Los bloques son listas de 0 o más instrucciones encerradas entre llaves ({}).

  • Los bloques se ejecutan siguiendo el mismo orden de ejecución de JavaScript.

  • Cuando dentro de un bloque se declara una variable con const o con let, solo existe dentro del bloque.

  • Una vez terminada la ejecución del bloque, todas las variables declaradas con const o con let soneliminadas.

  • Los bloques usan las variables o constantes declaradas fuera del bloque.

  • Las variables declaradas con const o con let, dentro de un bloque ocultan a otras con el mismo nombre que estén declaradas fuera del bloque.
