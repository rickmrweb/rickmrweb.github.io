El Elemento style

El elemento style debe colocarse en el elemento de tipo head y contiene el código CSS3 que define la apariencia de la página.

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport"
      content="width=device-width,
       initial-scale=1.0">
    <title>Hola Mundo</title>
    <!--
      Inicio del elemento de tipo
      style.
      Define la apariencia de la
      página.
    -->
    <style>
      body {
        color: white;
        background-color:
          rgb(0, 0, 255);
        font-family: serif;
      }
 
      h1 {
        color: #ffff00;
        font-family: sans-serif;
      }
    </style> <!--Termina style.-->
  </head>
  <body>
    <h1>Hola Mundo</h1>
    <p>Hola mundo.</p>
    <p>
      Este es mi primer ejemplo.
    </p>
    <footer>
      <p>
        &copy; 2020
        Gilberto Pacheco Gallegos.
      </p>
    </footer>
  </body>
</html>
