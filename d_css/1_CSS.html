<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport"
      content="width=device-width,
       initial-scale=1.0">
    <title>Hola Mundo</title>
    <style>
      body {
        color: white;
        background-color:
          rgb(0, 0, 255);
        font-family: serif;
      }

      h1 {
        color: #ffff00;
        font-family: sans-serif;
      }
    </style>
  </head>
  <body>
    <h1>Hola Mundo</h1>
    <p>Hola mundo.</p>
    <p>
      Este es mi primer ejemplo.
    </p>
    <footer>
      <p>
        &copy; 2020
        Gilberto Pacheco Gallegos.
      </p>
    </footer>
  </body>
</html>

Al código HTML se le agrega una etiqueta style, donde se colocan reglas y comentarios para configurar la apariencia de la página, usando un lenguaje conocido como CSS.

La página oficial de CSS3 está en https://www.w3.org/Style/CSS/

La sintaxis de CSS3 está en https://www.w3.org/TR/css-syntax-3/
