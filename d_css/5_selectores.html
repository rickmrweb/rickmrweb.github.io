Selector de Tipo

Una forma simple de selector es el selector de tipo, que consiste en usar el tipo de un elemento. Hace que una regla se aplique a todos los elementos de ese tipo.

La definición de una selector de tipo está en https://www.w3.org/TR/selectors-3/#type-selectors

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport"
      content="width=device-width,
       initial-scale=1.0">
    <title>Hola Mundo</title>
    <style>
      /* El selector "body" hace
       * que las propiedades de
       * esta regla se apliquen a
       * todos los elementos de
       * tipo "body". */
      body {
        color: white;
        background-color:
          rgb(0, 0, 255);
        font-family: serif;
      }

      /* El selector "h1" hace que
       * las propiedades de esta
       * regla se apliquen a todos
       * los elementos de tipo
       * "h1". */
      h1 {
        color: #ffff00;
        font-family: sans-serif;
      }
    </style>
  </head>
  <!--
    Las reglas con el selector
    "body" se aplican a este
    elemento.
  -->
  <body>
    <!--
      Las reglas con el selector
      "h1" se aplican a este
      elemento.
    -->
    <h1>Hola Mundo</h1>
    <p>Hola mundo.</p>
    <p>
      Este es mi primer ejemplo.
    </p>
    <footer>
      <p>
        &copy; 2020
        Gilberto Pacheco Gallegos.
      </p>
    </footer>
  </body>
</html>
