Reglas de Estilo

selector

Una expresión que permite identificar a uno o varios elementos de HTML.

lista de selectores

Una secuencia de selectores separadas por coma (,).

declaración de propiedad

Una expresión del tipo nombre: valor

lista de declaraciones de propiedades

Una secuencia de declaraciones de propiedades separadas por punto y coma (;) y encerradas entre llaves ({ }).

regla de estilo

Une lista de selectores con una lista de declaraciones de propiedades.

La definición de una regla de estilo está en https://www.w3.org/TR/css-syntax-3/#style-rule

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport"
      content="width=device-width,
       initial-scale=1.0">
    <title>Hola Mundo</title>
    <style>
      /* Esta es una regla.
       * El selector es body.
       * Sus propiedades están
       * dentro de las llaves. */
      body {
        color: white;
        background-color:
          rgb(0, 0, 255);
        font-family: serif;
      }/* Fin de la 1a regla. */

      /* Esta es otra regla.
       * El selector es h1.
       * Sus propiedades están
       * dentro de las llaves. */
      h1 {
        color: #ffff00;
        font-family: sans-serif;
      }/* Fin de la 2a regla. */
    </style>
  </head>
  <body>
    <h1>Hola Mundo</h1>
    <p>Hola mundo.</p>
    <p>
      Este es mi primer ejemplo.
    </p>
    <footer>
      <p>
        &copy; 2020
        Gilberto Pacheco Gallegos.
      </p>
    </footer>
  </body>
</html>
