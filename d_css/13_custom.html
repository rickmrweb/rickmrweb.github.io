Custom Properties

En CSS3 se pueden definir constantes y usar su valor a lo largo del código CSS.

Salida

Ábrelo en otra pestaña.

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport"
      content="width=device-width,
       initial-scale=1.0">
    <title>
      Custom Properties
    </title>
    <style>
      :root {
        --colorTexto: blue;
        --colorFondo: yellow;
      }
      h1 {
        color: var(--colorTexto);
        background-color:
         var(--colorFondo);
      }
      footer {
        color: var(--colorTexto);
      }
    </style>
  </head>
  <body>
    <h1>Custom Properties</h1>
    <p>Hola</p>
    <footer>
      <p>
        &copy; 2020
        Gilberto Pacheco Gallegos.
      </p>
    </footer>
  </body>
</html>
