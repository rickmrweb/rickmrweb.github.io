La Propiedad background-color

La propiedad background-color permite especificar el color del fondo del elemento.

La definición de la propiedad background-color está en https://www.w3.org/TR/css-backgrounds-3/#the-background-color

Salida

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport"
      content="width=device-width,
       initial-scale=1.0">
    <title>Hola Mundo</title>
    <style>
      body {
        color: white;
        /* Indica un color de
         * fondo azul para los
         * elementos tipo "body".
         */
        background-color:
          rgb(0, 0, 255);
        font-family: serif;
      }

      h1 {
        color: #ffff00;
        font-family: sans-serif;
      }
    </style>
  </head>
  <!--
    El color de fondo dentro de
    este elemento de tipo "body"
    es "rgb(0, 0, 255)".
  -->
  <body>
    <h1>Hola Mundo</h1>
    <p>Hola mundo.</p>
    <p>
      Este es mi primer ejemplo.
    </p>
    <footer>
      <p>
        &copy; 2020
        Gilberto Pacheco Gallegos.
      </p>
    </footer>
  </body>
</html>
