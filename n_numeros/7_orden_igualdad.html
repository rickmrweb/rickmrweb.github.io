Operadores combinados de orden e Igualdad

En el ejemplo que sigue, el orden natural es:

0 < 3 < 8

// Menor o igual que (<=)
console.log(3 <= 0); // false
console.log(3 <= 3); // true
console.log(3 <= 8); // true
// Mayor o igual que (>=)
console.log(3 >= 0); // true
console.log(3 >= 3); // true
console.log(3 >= 8); // false
