Consultas que se actualizan autom√°ricamente cada que hay un cambio.

Ejemplo

refDoc.onSnapshot(
  document => {
    if (document.exists) {
      console.log(document.id);
      const datos = document.data();
      console.log(datos);
    } else {
      console.log("No encontrado.");
    }
  },
