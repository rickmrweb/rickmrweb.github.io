Resumen

  • El tipo de datos boolean representa la lógica de valores verdaderos y falsos.

  • El tipo number representa números en JavaScript.

  • La estructura de control if permite usar el resultado de una condición lógica para controlar si un grupo de instrucciones debe o no ejecutarse.

    Si la condición evalúa a true, el grupo de instrucciones se ejecuta.

    Si la condición evalúa a false, el grupo de instrucciones no se ejecuta.

  • La instrucción const con = require("readline-sync"); asigna a la constante con un valor que permite pedir y leer datos en la consola.

  • La instrucción const fruta = teclado.question("¿Cuál es tu fruta favorita?"); muestra el texto ¿Cuál es tu fruta favorita? en la consola, espera a que el usuario teclee su respuesta, la cuál se asigna a la variable fruta.

  • Antes de empezar a programar, recuerda estudiar bien el problema, diseñar algunos escenarios y la solución, usando pseudocódigo o diagramas de flujo.

  • Cuando pruebes una aplicación, recuerda verificar los escenarios.

  • Los programas se pueden crear y ejecutar en internet usando Replit.

  • Los programas se pueden crear y ejecutar en tu computadora usando Visual Studio Code.

  • Las aplicaciones que piden datos al usuario se pueden depurar con Visual Studio Code.
