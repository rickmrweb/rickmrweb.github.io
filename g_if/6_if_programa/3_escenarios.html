Escenarios

Los escenarios son ejemplos con datos para mostrar el comportamiento que debe tener el programa.

Cuando el programa esté listo, se puede probar introduciendo los datos de entrada de cada escenario y el programa debe generar la salida correspondiente.

Cuando los programas usan consola, imagína un diálogo en cuál el programa inicia preguntando, luego espera a que el usuario conteste y finalmente el programa muestra resultados.

Escenario: naranja

Programa:
¿Cuál es tu fruta favorita?
Usuario:
naranja
Programa:
Tu fruta: naranja
