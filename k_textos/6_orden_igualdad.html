Operadores Combinados de Orden e Igualdad

En el ejemplo que sigue, el orden natural es:

"aa" < "ah" < "ai"

// Menor o igual que (<=)
console.log("ah" <= "aa"); // false
console.log("ah" <= "ah"); // true
console.log("ah" <= "ai"); // true
// Mayor o igual que (>=)
console.log("ah" >= "aa"); // true
console.log("ah" >= "ah"); // true
console.log("ah" >= "ai"); // false
