El Elemento html

El elemento html representa una página. Es el elemento principal. Solo puede haber uno de estos elementos en un archivo.

Su especificación está en https://html.spec.whatwg.org/multipage/semantics.html#the-html-element

El elemento body tiene 2 elementos hijos: head y body.

<!DOCTYPE html>
<!--
  Esta es la etiqueta inicial de
  los archivos html.
  Tiene 2 elementos hijos: head y
  body.
  lang="es"
    lang
      indica el idioma de la
      página.
    "es"
      indica que el idioma de la
      página es español.
-->
<html lang="es">
  <!--
    Este head es elemento hijo de
    html.
  -->
  <head>
    <meta charset="UTF-8">
    <meta name="viewport"
      content="width=device-width,
       initial-scale=1.0">
    <title>Hola Mundo</title>
  </head>
  <!--
    El otro elemento hijo de html
    es body.
  -->
  <body>
    <h1>Hola Mundo</h1>
    <p>Hola mundo.</p>
    <p>
      Este es mi primer ejemplo.
    </p>
    <footer>
      <p>
        &copy; 2020
        Gilberto Pacheco Gallegos.
      </p>
    </footer>
  </body>
</html><!-- Termina html. -->
