Elementos

Un elemento es un objeto que representa una forma de procesar el texto y se escribe como una etiqueta.

Cada elemento tiene asociado un tipo, que representa su forma de procesamiento.

Cada tipo de elemento tiene un funcionamiento preestablecido por el navegador.

El tipo es la primera palabra de la etiqueta.

<!DOCTYPE html>
<!-- Elemento de tipo html. -->
<html lang="es">
  <!-- Elemento de tipo head. -->
  <head>
    <!-- Elemento tipo meta. -->
    <meta charset="UTF-8">
    <!-- Elemento tipo meta. -->
    <meta name="viewport"
      content="width=device-width,
       initial-scale=1.0">
    <!-- Elemento tipo title. -->
    <title>Hola Mundo</title>
  </head>
  <!-- Elemento de tipo body. -->
  <body>
    <!-- Elemento de tipo h1. -->
    <h1>Hola Mundo</h1>
    <!-- Elemento de tipo p. -->
    <p>Hola mundo.</p>
    <!-- Elemento de tipo p. -->
    <p>
      Este es mi primer ejemplo.
    </p>
    <!-- Elemento tipo footer. -->
    <footer>
      <!-- Elemento de tipo p. -->
      <p>
        &copy; 2020
        Gilberto Pacheco Gallegos.
      </p>
    </footer>
  </body>
</html>
