El Elemento head

El elemento head define características para procesar el archivo.

Es uno de los elementos hijos de html.

Su especificación está en https://html.spec.whatwg.org/multipage/semantics.html#the-head-element

<!DOCTYPE html>
<html lang="es">
  <!--
    El elemento head define
    características de la página y
    es uno de los elementos hijos
    de html.
  -->
  <head>
    <meta charset="UTF-8">
    <meta name="viewport"
      content="width=device-width,
       initial-scale=1.0">
    <title>Hola Mundo</title>
  </head><!-- Termina head. -->
  <body>
    <h1>Hola Mundo</h1>
    <p>Hola mundo.</p>
    <p>
      Este es mi primer ejemplo.
    </p>
    <footer>
      <p>
        &copy; 2020
        Gilberto Pacheco Gallegos.
      </p>
    </footer>
  </body>
</html>
