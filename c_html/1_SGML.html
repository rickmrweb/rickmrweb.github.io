<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport"
      content="width=device-width,
       initial-scale=1.0">
    <title>Hola Mundo</title>
  </head>
  <body>
    <h1>Hola Mundo</h1>
    <p>Hola mundo.</p>
    <p>
      Este es mi primer ejemplo.
    </p>

      <p>
        &copy; 2020
        Gilberto Pacheco Gallegos.
      </p>
    </footer>
  </body>
</html>

El código está escrito en un lenguaje llamado SGML, que consta de contenido y marcas (también conocidas como etiquetas).

Contenido

Es el texto se va a mostrar.

Marcas o Etiquetas

Se le añaden al contenido.

Inician con el caracter < y terminan con >.

Indican como desplegar el texto; por ejemplo, el texto entre <h1> y </h1> resalta más.

También se puede ver que el texto entre <p> y </p> aparece en un párrafo, separado del resto del texto.
