El Metadato viewport

El metadato viewport indica como manejar la pantalla en navegadores de dispositivos móviles.

Si no se pone esta etiqueta y la página se muestra en un navegador móvil, se muestra reducida en tamaño.

Puedes encontrar más información en https://developer.mozilla.org/es/docs/M%C3%B3vil/Viewport_meta_tag

Salida

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <!--
      meta
        Información sobre la
        página.
      name="viewport"
        Define el metadato
        viewport.
      content="width=device-width,
       initial-scale=1.0"
        Es el valor asociado con
        el metadato viewport, que
        consta de varios atributos.
      width=device-width
        El ancho de la página será
        el ancho del dispositivo,
        lo que permite mostrar las
        letras del tamaño normal
        para el móvil.
      initial-scale=1.0
        Inicialmente, la página se
        muestra sin modificar el
        tamaño natural de la
        página; es decir, que ni
        se amplifica ni se reduce
        el contenido.
    -->
    <meta name="viewport"
      content="width=device-width,
       initial-scale=1.0">
    <title>Hola Mundo</title>
  </head>
  <body>
    <h1>Hola Mundo</h1>
    <p>Hola mundo.</p>
    <p>
      Este es mi primer ejemplo.
    </p>
    <footer>
      <p>
        &copy; 2020
        Gilberto Pacheco Gallegos.
      </p>
    </footer>
  </body>
</html>
